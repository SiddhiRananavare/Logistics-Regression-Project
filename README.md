# Logistics-Regression-Project
Implemented Logistic Regression to classify data based on input features. Trained and evaluated the model using metrics such as accuracy, precision, recall, and confusion matrix. Optimized performance using feature selection and preprocessing techniques.

📌 Tumor Detection using Logistic Regression

🔹 Overview

-This project focuses on building a classification model to detect whether a tumor is benign or malignant using Logistic Regression. The goal is to assist in early diagnosis by analyzing medical data and predicting tumor type.

🔹 Objectives

-Classify tumors as benign or malignant

-Build a reliable classification model

-Evaluate model performance using classification metrics

🔹 Dataset Description

-The dataset contains medical features such as cell size, texture, radius, and other characteristics extracted from tumor samples. The target variable indicates whether the tumor is benign (0) or malignant (1).

🔹 Key Tasks Performed

-Data cleaning and preprocessing

-Handled missing values and feature scaling

-Performed exploratory data analysis (EDA)

-Split dataset into training and testing sets

-Trained Logistic Regression model

-Predicted tumor classes (benign/malignant)

-Evaluated model using accuracy, precision, recall, and confusion matrix

🔹 Tools & Technologies
-Python

-Pandas

-NumPy

-Scikit-learn

-Matplotlib / Seaborn

🔹 Evaluation Metrics
-Accuracy

-Precision

-Recall

-F1 Score

-Confusion Matrix

🔹 Results

-The Logistic Regression model achieved strong classification performance and was able to accurately distinguish between benign and malignant tumors.

🔹 Conclusion

-This project demonstrates how Logistic Regression can be effectively used for medical classification tasks, helping in early detection and better decision-making in healthcare.
